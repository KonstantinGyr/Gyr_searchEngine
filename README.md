        # **Gyr_searchEngine**
        ## Описание
   Поисковый движок **Gyr_searchEngine** -консольное приложение, которое по введенному запросу сортирует 
существующие документы по релевантности и записывает их в json формате.
   При запуске Gyr_searchEngine проверяет файл конфигурации и выводит в консоль название движка и его
текущую версию. Далее программа предлагает пользователю загрузить документы, по которым будет осущест-
вляться поиск, в специальную папку и предложит задать количество документов выводимых на один запрос.
